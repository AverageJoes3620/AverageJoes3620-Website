Colophon

The website for the Average Joes was written by the web subteam which was mentored by Christoher Engelhardt.

Backend

The website was written in valid HTML5 and CSS3. The team collaborated on GitHub, the repository is viewable here. The website is hosted by Nuqnet through one of our sponsors, Working Zombie Studios.

Design

The web subteam designed and wrote the template. Some of the website's content was written by other team members while a majority was written by the web subteam.

The colors used in the website are the official school colors for the Saint Joseph Public Schools. The hexidecimal color for the blue is: #002663, and the yellow is: #FFD200. The body copy is in the font Roboto Light 300, which is availble from Google Fonts. The headings and titles are in Oswald Bold 700, which is availble from Google Fonts.
